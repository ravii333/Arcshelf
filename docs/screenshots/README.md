# Screenshots

Place PNG screenshots here using these exact filenames so the main README renders them:

- `home.png` — landing page (`/`)
- `browse.png` — browse papers (`/browse`)
- `paper-detail.png` — a single paper with the PDF viewer (`/questions/:id`)
- `submit.png` — the upload form (`/submit`, requires login)
- `admin.png` — moderation dashboard (`/admin`, requires an admin account)
- `dashboard.png` — user dashboard / contributions (`/dashboard`, requires login)

Recommended: 1440×900 viewport, PNG, and crop out browser chrome.
